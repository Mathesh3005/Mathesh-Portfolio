# 💼 Mathesh S — Portfolio
A modern, dark-themed personal portfolio website showcasing my skills, projects, and experience as a Full Stack Developer.
🔗 **Live Preview**: [Open `index.html`](index.html)
---
## ✨ Features
- 🌑 **Dark Glassmorphism Design** — Premium translucent cards with gradient accents
- ⌨️ **Typing Animation** — Dynamic hero subtitle cycling through roles
- 🔵 **Particle Background** — Animated connected dots in the hero section
- 🎞️ **Scroll Animations** — Smooth fade-in effects on all sections via Intersection Observer
- 📱 **Fully Responsive** — Mobile-first layout with hamburger navigation (down to 375px)
- 🔍 **SEO Optimized** — Semantic HTML, meta tags, and proper heading hierarchy
